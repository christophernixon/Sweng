A visualization of some of the data from the Github API

This is an application for visualising data taken from the github API, with a backend written in python (Flask) and a sample front end using a Circularpacking chart example.

Circular packing allows one to visualize a hierarchic organization. It is an equivalent of a treemap or a dendrogram,where each node of the tree is represented as a circle and its sub-nodes are represented as circles inside of it.
